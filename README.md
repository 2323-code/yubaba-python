# yubaba-python

## とは

Python のコンソール上で湯婆婆を実装したものです。
友人らへのチュートリアル用として作成しました。

## 環境構築

1. `make up` で Python のコンテナを立ち上げます。
2. 完了したら、`make app` で コンテナに入ります。
3. コンテナのターミナルが起動するので、その中で `python yubaba.py`と実行すると、コードが実行されます。

文言は一部学習のために変更しています。
